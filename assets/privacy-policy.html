D2 Mule Tracker — Privacy Policy

Last updated: April 2026

Overview

D2 Mule Tracker is an offline inventory tracking app for Diablo 2 Resurrected. Your privacy is important to us.

Data Collection

We do not collect, store, transmit, or share any personal data.

All data you enter into the app (character names, item inventories, notes) is stored locally on your device only. No data is sent to any server, cloud service, or third party.

Analytics & Tracking

This app does not include any analytics, crash reporting, advertising SDKs, or tracking of any kind.

Network Usage

This app does not require an internet connection and makes no network requests during normal use. The item database is bundled with the app at install time.

Data Storage

All user data is stored in a local SQLite database on your device. You can export your data as a JSON file using the app's built-in export feature, and import it on another device using the import feature. These operations use your device's native sharing capabilities.

Data Deletion

Uninstalling the app removes all stored data. You can also delete individual realms, characters, or items within the app at any time.

Children's Privacy

This app does not knowingly collect information from children under 13. Since no data is collected from any user, this is not applicable.

Changes to This Policy

If we update this privacy policy, the changes will be posted here with an updated date.
